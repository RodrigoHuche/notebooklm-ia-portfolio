# notebooklm-ia-portfolio
Projeto desenvolvido para o desafio da DIO utilizando o NotebookLM como ferramenta de aprendizagem ativa, com curadoria de fontes, engenharia de prompts e um miniguia de estudos sobre Inteligência Artificial.

 Miniguia de Estudos com NotebookLM – Inteligência Artificial para Profissionais de TI
 Contexto

Este projeto foi desenvolvido como parte do desafio da DIO com o objetivo de explorar o potencial do NotebookLM como ferramenta de aprendizagem ativa.

O foco do estudo foi compreender os principais conceitos de Inteligência Artificial, Machine Learning, IA Generativa e Engenharia de Prompts utilizando fontes confiáveis.

Além de gerar resumos, o NotebookLM foi utilizado para criar explicações, comparar conceitos, responder perguntas e organizar um guia de revisão.

 Objetivos
Aprender os conceitos fundamentais de IA.
Entender as diferenças entre IA Tradicional, Machine Learning, Deep Learning e IA Generativa.
Criar um material organizado para futuras revisões.
Desenvolver habilidades de Engenharia de Prompts.
Aprender a utilizar o NotebookLM como ferramenta de estudos.

 Engenharia de Prompts

Durante o estudo foram realizados diversos testes de prompts.

Prompt 1

Explique Inteligência Artificial como se estivesse ensinando alguém que trabalha com suporte técnico.

Resultado:

Resposta clara e adaptada ao contexto profissional.

Prompt 2

Compare IA Tradicional, Machine Learning, Deep Learning e IA Generativa em formato de tabela.

Resultado:

Excelente organização das diferenças entre as tecnologias.

Prompt 3

Crie um resumo em tópicos sobre Machine Learning utilizando apenas as fontes fornecidas.

Resultado:

Resposta objetiva e baseada exclusivamente nas referências.

Prompt 4

Gere 10 perguntas para revisar o conteúdo.

Resultado:

Ótimo para fixação do aprendizado.

Prompt 5

Explique o conteúdo utilizando exemplos do dia a dia de um técnico de informática.

Resultado:

Foi o prompt que gerou a melhor compreensão prática.

 Troubleshooting (Cicatrizes)
Problema 1

A IA começou a responder utilizando conhecimento externo.

Solução

Adicionar ao prompt:

Utilize apenas as fontes fornecidas.

Problema 2

Respostas muito longas.

Solução

Adicionar:

Responda em até 10 linhas.

Problema 3

Respostas superficiais.

Solução

Solicitar:

Explique passo a passo com exemplos práticos.

 Miniguia de Estudos
O que é Inteligência Artificial?

É uma área da computação responsável por desenvolver sistemas capazes de realizar tarefas que normalmente exigiriam inteligência humana.

Exemplos:

reconhecimento de imagens
tradução automática
assistentes virtuais
chatbots
análise de dados
Machine Learning

É um ramo da IA onde os sistemas aprendem por meio dos dados.

Fluxo:

Dados

Treinamento

↓

Modelo

↓

Previsão

Deep Learning

É uma evolução do Machine Learning baseada em Redes Neurais Artificiais.

É utilizado em:

reconhecimento facial
carros autônomos
processamento de linguagem natural
visão computacional
IA Generativa

É uma IA capaz de criar conteúdo novo.

Exemplos:

textos
imagens
músicas
vídeos
códigos

NotebookLM

O NotebookLM é uma ferramenta da Google que permite estudar documentos utilizando Inteligência Artificial.

Principais recursos:

criação de resumos
geração de perguntas
mapas mentais
organização do conhecimento
respostas baseadas apenas nas fontes fornecidas

Glossário

 Aprendizados

Durante este projeto foi possível compreender que o NotebookLM vai além de um simples chatbot.

Seu maior diferencial é produzir respostas fundamentadas nas fontes adicionadas pelo usuário, tornando-o uma excelente ferramenta para estudos, pesquisas e organização do conhecimento.
